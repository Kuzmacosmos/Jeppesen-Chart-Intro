# Jeppesen-Chart-Intro
一份通俗易懂的杰普逊航图识读手册，源自个人B站专栏。介于B站和百度云会吞各种连接，我将所有资源留存于此，方便粉丝和读者参考。
### A Great Update
CAAC eAIP已经采用全新的航图版面设计，并在四月出版周期中陆续更换。因此本手册也会在近期同步更新，旧版航图识图部分保留，供读者对比参照。
